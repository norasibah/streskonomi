Pemantau Streskonomi

Sila jawab semua soalan di bawah sebelum melihat keputusan.

A. Kewangan

  1. Saya rasa tertekan untuk membayar bil bulanan tepat pada masanya.
  2. Saya rasa risau jika pendapatan bulanan tidak mencukupi untuk keperluan asas.
  3. Saya rasa bimbang tentang insiden kenaikan harga barang.
  4. Saya rasa tertekan dengan perbelanjaan harian yang semakin meningkat.
  5. Pengurusan bajet saya sering negatif.

B. Pendapatan dan Pekerjaan

  1. Saya bimbang kehilangan pekerjaan atau pendapatan.
  2. Pendapatan saya tidak setimpal dengan kos sara hidup yang lebih tinggi.
  3. Keadaan ekonomi yang tidak menentu membuat saya risau.
  4. Saya tertekan apabila memikirkan masa depan kerjaya saya.
  5. Saya merasa risau tentang kewangan diwaktu kecemasan yang tidak dijangka.

C. Simpanan & Kesediaan Masa Depan

  1. Saya merasa bimbang kerana tidak mempunyai simpanan yang mencukupi.
  2. Saya takut tidak mampu menampung kos pendidikan anak pada masa-masa akan datang.
  3. Saya risau tentang keupayaan kewangan semasa menghadapi persaraan.
  4. Saya tidak yakin dengan kestabilan kewangan keluarga pada masa depan.

D. Hutang & Komitmen Kewangan

  1. Saya tertekan dengan jumlah hutang yang perlu dibayar.
  2. Saya sukar mengimbangi bayaran hutang dengan perbelanjaan kehidupan sehari-hari.
  3. Saya risau jika gagal membayar pinjaman tepat pada waktunya.
  4. Hutang memberi kesan negatif kepada kesejahteraan mental saya.

E. Tekanan Ekonomi Sosial

  1. Saya berasa malu kerana tidak mencapai tahap standard hidup masyarakat lain.
  2. Saya tertekan apabila perlu membantu kewangan keluarga.
  3. Saya tertekan apabila melihat orang lain mengalami status kewangan yang lebih stabil.
  4. Saya berasa rendah diri kerana kedudukan kewangan saya tidak mencapai standard seperti orang lain.

F. Reaksi Emosi & Psikologi

  1. Saya mudah marah kerana masalah kewangan.
  2. Masalah kewangan mengganggu tidur saya.
  3. Saya cepat cemas ketika memikirkan masa depan ekonomi.
  4. Saya rasa hilang kawalan terhadap kewangan.
  5. Saya berasa putus asa tentang situasi ekonomi saya.
